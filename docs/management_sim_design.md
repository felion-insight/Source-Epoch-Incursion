## 二、模拟经营设计文档（完整版）

### 模拟经营系统设计文档

**版本**：1.0  
**关联游戏**：AI叙事 + 模拟经营游戏（阴谋扩展版）  
**核心原则**：轻量化、叙事驱动、决策即剧情

### 1. 设计目标

- 为剧情对话提供**结构性选择**，而非增加复杂度。
- 每个经营决策都应产生明显的叙事后果，并改变玩家与NPC的关系。
- 资源管理应服务于**道德困境和势力站队**，而非生存挑战。
- 最终，经营系统可被自动化，让剧情爱好者聚焦对话。

### 2. 资源系统（4种核心资源）

| 资源 | 描述 | 获取方式 | 主要消耗 | 枯竭后果（叙事向） |
|------|------|----------|----------|-------------------|
| **能源（源）** | 基地运行的基础，由源矿机产出 | 源矿机（随时间衰减）、**地图矿区主动开采**（见 Tech 规格 §4）、探索、交易、远征结算 | 维持防御、升级建筑、解锁高级对话 | 防御崩溃→NPC受伤/死亡→绝望剧情分支 |
| **食物** | 养活基地人口 | 农业模块、**岸线/防务后勤跑腿**（见 Tech 规格 §4）、狩猎队、交易 | 日常消耗 | 饥饿爆发→NPC互相指控→玩家需选择优先供应对象 |
| **医疗物资** | 治疗伤病、稳定同调副作用 | 生产、**实验室内短时配给玩法**（见 Tech 规格 §4）、探索、与净空会交换 | 治疗伤员、维持NPC健康 | 伤员死亡→NPC因痛苦向其他势力求助→可能的背叛 |
| **情报点** | 揭示阴谋的关键资源 | 升级通讯阵列、**频段/信道类地图交互**（见 Tech 规格 §4）、解码、与特定NPC私通、远征归国 | 解锁真相、联络势力 | 无法获得真相→走向盲目结局 |

### 3. 经营模块（5个核心设施）

每个设施仍以**少量强分支决策**为核心（表中各选项仍为真相源）；**可视化上的「升级树」**用于表达前置顺序与互斥组，深度的上限与 DAG 字段见 **[sim_facility_tech_and_resource_gameplay.md](./sim_facility_tech_and_resource_gameplay.md)**。  
本节不赘述树节点 CSV——避免与程序化后的 JSON 双龙维护。

#### 3.1 通讯阵列

- **基础功能**：与其他基地交易资源（每交易一次，获得少量任意资源）
- **升级选项**：
  - 加密破解模块（消耗能源20，情报 +10）→ 破译议会密文
  - 全频广播系统（消耗能源30，情报 +5）→ 可向全球广播
- **叙事绑定**：加密模块安装后，回声-7首次联系玩家。

#### 3.2 源矿机

- **基础功能**：自动产出能源（速率随时间下降，需维护）
- **管理模式**：
  - 加大开采深度 → 能源 +40，岸线侵入 +20%
  - 限制开采 → 能源 -20，岸线侵入 -10%
  - 维持现状 → 无变化
- **叙事绑定**：加大开采后，林博士紧急警告，出现矿区塌方事件。

#### 3.3 医疗/研究实验室

- **基础功能**：治疗伤病、研发药物
- **升级选项**：
  - 神经扫描仪（消耗医疗15，情报 +15）→ 检测NPC体内监控芯片
  - 同调抑制器（消耗医疗25，科研 +10）→ 降低玩家同调值
- **叙事绑定**：神经扫描仪可发现卡伦的秘密。

#### 3.4 防御工事

- **基础功能**：延缓岸线侵入推进速度
- **等级**：薄弱（默认）、标准、坚固
- **升级消耗**：能源20→标准，再30→坚固
- **叙事绑定**：坚固等级触发回声集团渗透事件。

#### 3.5 地下监听站（特殊，中后期解锁）

- **解锁条件**：获得小胖或林博士提供的线索
- **功能**：聆听“源的低语”，获得大量情报，但大幅提升玩家同调值
- **建造消耗**：能源50
- **叙事绑定**：建造后，回声-7试图干扰监听。

### 4. 经营决策如何驱动剧情（映射机制）

- **决策 → 资源变化 → 触发预定义事件 → 改变NPC记忆/信任值 → 解锁新对话选项**
- 示例流程：
  - 玩家选择“安装加密破解模块” → 能源 -20，情报 +10 → 自动破译议会密文 → 卡伦紧张 → 下次对话出现“我知道你是灰镜”选项。

### 5. 岸线侵入系统（动态压力）

- **可视化**：地图上红色边界，每推进一格，玩家活动区域缩小。
- **推进条件**：
  - 每5小时自动推进一格（游戏内时间）。
  - 玩家“加大开采深度”时，额外加速20%推进。
- **推进后果**：
  - 失去某些NPC（撤离或死亡）。
  - 解锁撤离对话。
  - 若推进到基地核心，触发强制撤离结局。
- **减速手段**：升级防御工事、限制开采、与净空会合作引爆源矿。

### 6. 委任系统（聚焦剧情模式）

- **触发**：玩家可选择“委任卡伦”或“委任林博士”代管基地。
- **效果**：
  - 卡伦：优先升级防御，限制开采 → 能源较低，但岸线侵入慢。
  - 林博士：优先升级实验室，加大开采 → 情报多，但岸线侵入快。
- **后果**：被委任的NPC信任值上升，但玩家无法控制具体决策。可随时收回管理权。

### 7. 资源获取简化设计

- **交易系统**：通过通讯阵列，每7天可进行一次批量交易（如用10情报换20能源）。
- **探索任务**：派遣小胖或卡伦外出（消耗时间），带回随机资源，同时触发野外剧情事件；与本仓库已实现之**远征占位**同属「委托异步」一脉。
- **地图主动循环**：玩家在已解锁区域内完成短时操作换取指定资源（与 tick / 决算并行）；区域与四类资源默认值见 **[sim_facility_tech_and_resource_gameplay.md](./sim_facility_tech_and_resource_gameplay.md) §4**。
- **对话奖励**：帮助某个NPC解决问题，可获得其私人储藏资源（如食物或医疗包）。

### 8. 界面与交互建议

- **资源显示**：右上角常驻4种资源图标与数值，紧缺时变红闪烁。
- **经营面板**：基地地图模式，点击建筑弹出升级/决策菜单。
- **事件通知**：当决策触发剧情事件时，弹出对话框，玩家可选择立即观看或稍后回顾。
- **快捷委任**：主界面设置“委任模式”开关，一键切换。

### 9. 与AI NPC系统的结合点

- **NPC会根据当前资源状况生成不同对话**：
  - 能源短缺 → 卡伦：“我们需要更多源，否则防御会崩溃。”
  - 食物充足 → 小胖：“今天有肉吃！心情好多了。”
- **NPC的信任值变化会直接反馈在AI生成的语言风格中**（如高信任值时卡伦会用更亲切的语气）。
- **经营决策如果伤害到某个NPC的利益**（如拒绝回声集团援助导致监听站被入侵），该NPC会主动找玩家谈话，表达不满或请求改变。

#### 9.1 经营决策的AI个性化反馈（替代部分预设事件）

当玩家完成一项**经营决策**后（如升级、分配资源、拒绝势力），系统会随机选择1-2名关联NPC，根据其性格、当前信任值及对决策的态度，实时生成一段评论对话（**非预设文本**）。

**触发时机**：决策立即生效后，或进入基地场景时通过弹出气泡/私信形式展示。

**示例**：

玩家选择“加大开采”，林博士（信任值低）：“你……你怎么能这样？我见过这种贪婪带来的后果。”

玩家选择“接受回声集团援助”，卡伦（信任值中）：“我不同意，但我会服从命令。不过别指望我协助他们。”

经济性控制：每个经营决策最多触发2条评论，且每天最多触发3次，避免API过载。

#### 9.2 经营系统触发生成临时NPC

在特定经营决策后（如“建造地下监听站”、“升级通讯阵列”），有概率吸引临时NPC访问基地。**例如**：

监听站建成后，可能吸引一名“流浪科学家”前来求宿，玩家可与其对话（全AI生成），获取随机情报点或触发小型支线。

防御工事升级后，可能有“难民商人”出现在海岸线，交易特殊物品（如一份旧日志碎片，指向时间重置真相）。

这些临时NPC由AI完全生成，拥有简单记忆（对话内容），离开后消失。他们不会影响主线，但能强化“世界是活的”感受。

### 10. 设计总结

- 经营系统不做数值平衡游戏，而是**叙事分支的触发器**。
- **「树 + 地图主动循环」扩充可玩性**，但不改变：`management` tag 账本、Silent 闸门、主线真相投递规则（仍以上游 sim_* 文档为准）。
- 4种资源、5个模块（及扩展设施位）、岸线侵入进度、委任系统，构成简洁但有效的经营层。
- 玩家花费在经营上的时间应远小于对话，且每个经营操作都应有明确的叙事回报。
- 最终，经营历史记录作为隐藏变量，参与结局计算。

*本设计可与阴谋文档、人物设定文档配合使用。*
