## 图例说明

| 符号 | 含义 |
|------|------|
| `[名称]` | 可探索地点 |
| `↑ ↓ ← →` | 移动连接方向（可原路返回） |
| `⛔ 剧情锁` | 需完成特定主线任务/获得关键物品 |
| `⛔ 信息锁` | 需从NPC或文档获得坐标线索 |
| `⛔ 能力锁` | 需特定装备（如探照灯） |
| `⛔ 时间锁` | 需特定时刻（如夜间） |
| `⛔ 资源锁` | 需消耗能源等资源建造/进入 |

## 与各资源「地图主动玩法」的对应（提要）

大世界几何与门锁仍以**上表与本节前文**为准。四类核心资源首推在哪些区域做多做「亲力亲为」的可重复小额产出，默认值与冷却/岸线约束详见 **[sim_facility_tech_and_resource_gameplay.md](./sim_facility_tech_and_resource_gameplay.md) §4**；实现时请令 `explorer_access` 的 `zone_id` 与本表「区域」及代码中的设施 slug 对齐。

---

此地图设计确保了玩家按**序章 → 第一幕 → 第二幕 → 第三幕**顺序解锁区域，无法提前接触后期关键地点。如需更详细的场景内部结构（如沉没实验室有3层），可后续补充。
