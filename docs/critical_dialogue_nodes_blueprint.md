<a id="一节点表使用说明"></a>## 一、节点表使用说明

本表仅包含__关键剧情节点__（约40\-50个），每个节点定义：

- __必达信息__：AI生成时必须传达的核心内容（防止偏离）
- __分支方向__：玩家可选的大类方向（2\-4个）
- __变量影响__：选择后影响的隐藏变量及数值
- __后续解锁__：开启哪些新节点或支线

非关键节点（日常寒暄、临时NPC对话、部分可选调查）由AI自由生成，不受本表约束。

__节点编号规则__：幕次\(章\)\-序号，例如 01\-03 表示第一幕第3节点。

<a id="三节点详细规格选代表性节点详述"></a>## 三、节点详细规格（选代表性节点详述）

<a id="pro-01-抵达基地"></a>### PRO\-01 抵达基地

- __场景__：直升机停机坪，阴雨  

- __参与NPC__：卡伦  

- __必达信息__：
	- 基地刚刚经历一次小型岸线侵入，损失惨重
	- 玩家被紧急召回担任代理指挥官
	- 前任指挥官已殉职
- __分支选项__：
	- A\. “我会尽力” → HUMAN \+2，卡伦信任\+5
	- B\. “我只是临时顶替” → 无变化
	- C\. “谁干的？我要查清楚” → 怀疑值\+5（隐藏），卡伦信任\-5
- __后续解锁__：PRO\-02

<a id="升级优先第一次经营决策"></a>### 01\-02 升级优先（第一次经营决策）

- __场景__：基地会议室（地图界面）  

- __参与NPC__：无（玩家独自选择）  

- __必达信息__：列出三个可升级设施及其预算影响  

- __分支选项__：
	- A\. 优先升级通讯阵列 → 解锁回声\-7线，COUNCIL\-5
	- B\. 优先升级源矿机 → 短期能源\+20，INCURSION\+5
	- C\. 优先升级医疗实验室 → 解锁扫描仪，HUMAN\+5
- __后续解锁__：根据选择，后续节点不同（如选A，则01\-05回声\-7提前出现）

<a id="卡伦的身份暴露"></a>### 02\-05 卡伦的身份暴露

- __场景__：监控室或通讯阵列  

- __参与NPC__：卡伦，可能包含林博士  

- __必达信息__：
	- 卡伦是议会安插的“灰镜”
	- 她一直在监控玩家的同调值
	- 她有权执行“清理协议”
- __触发方式__（前序决定）：
	- 方式1：玩家通过神经扫描仪发现（若升级了医疗实验室）
	- 方式2：玩家质问（若获得足够情报）
	- 方式3：卡伦主动坦白（高信任值下）
- __分支选项__（以方式2为例）：
	- A\. 当场逮捕卡伦 → 卡伦被囚禁，COUNCIL\-20，失去后续帮助
	- B\. “我理解你是被迫的” → 卡伦感动，信任\+20，KAREN\_TRUST设为True，后续提供议会密文
	- C\. “继续为我工作，但我会利用你” → 卡伦继续监控但暗中提供假数据，ECHO\+10（回声集团可渗透），信任\-10
- __后续解锁__：02\-06（卡伦的结局变体）

<a id="第一次站队"></a>### 02\-08 第一次站队

- __场景__：基地核心大厅，三方代表同时出现（全息/对话）  

- __参与NPC__：伊丽莎白·莫罗（全息），回声\-7（屏幕），堇（真人）  

- __必达信息__：
	- 议会：要求玩家继续配合筛选，承诺“人类未来”
	- 净空会：要求玩家协助引爆源矿，牺牲部分区域
	- 回声集团：要求玩家开放基地主控，接受神经统一
- __分支选项__：
	- A\. 加入议会 → COUNCIL\+30，PURIFY\-20，ECHO\-20
	- B\. 加入净空会 → PURIFY\+30，COUNCIL\-20，ECHO\-20
	- C\. 加入回声集团 → ECHO\+30，COUNCIL\-20，PURIFY\-20
	- D\. 拒绝所有，保持独立 → 各势力好感度均\-10，但HUMAN\+15，INSIGHT\+10（解锁隐藏结局条件）
- __后续解锁__：根据选择，第三章部分节点内容变化（例如加入议会后，克莱因可能拒绝见面）

<a id="克莱因的遗言"></a>### 03\-02 克莱因的遗言

- __场景__：深层囚室，昏暗，克莱因躺在病床上  

- __参与NPC__：克莱因  

- __必达信息__（克莱因将在4段对话中依次揭示，每段后可能咳嗽中断）：
	- 源是上个宇宙轮回中先驱文明的全部记忆
	- 源纪元实际已经持续230年，人类记忆被重置多次
	- 玩家是唯一的“双向同调者”，可以改写源
	- 先驱遗言：不要成为源，保留个体意识
- __分支选项__（每段揭示后出现）：
	- 揭示第一段后：A\. 相信他 / B\. 怀疑他 → 影响INSIGHT
	- 揭示第二段后：A\. 追问时间重置细节 / B\. 问他为何被囚禁
	- 揭示第三段后：A\. 接受命运 / B\. 拒绝相信自己是特殊
	- 揭示第四段后：A\. 承诺解救他 / B\. 询问如何终结源
- __重要限制__：克莱因最多讲述4段，之后会死亡（无论玩家是否问完）。每一段消耗一次“揭露计数”，超过4次强制死亡。
- __变量影响__：每个选择的组合影响INSIGHT和HUMAN。

<a id="源深度对话"></a>### 03\-03 源深度对话

- __场景__：地下监听站，屏幕波纹，无视觉形象  

- __参与NPC__：源（通过自由文本输入）  

- __必达信息__：无。源不会直接告知任何事实，只会提供记忆碎片和隐喻。
- __交互方式__：玩家输入任何文字，源根据同调值、INSIGHT和之前对话历史生成1\-3句回应。
- __AI生成约束__：
	- 不能直接回答“是/否”事实问题
	- 必须使用非人类语法（如“我们…记得…光…熄灭…”）
	- 如果玩家同调值<30，回答极模糊；>70，回答相对清晰但依然诗意
- __重要分支__：玩家的提问态度会被记录（例如“你痛苦吗？” → 同情；“你能给我力量吗？” → 功利），影响INSIGHT和结局8。
- __后续解锁__：若玩家连续3次提问表现出对源的深刻理解（INSIGHT>70），则触发03\-04先驱遗言（直接获得，无需克莱因）。

<a id="fin-02-最终选择"></a>### FIN\-02 最终选择

- __场景__：无特定，依据结局不同（废墟/服务器机房/源矿核心等）
- __参与NPC__：无（玩家独自面对）  

- __必达信息__：根据当前变量，系统列出__可达结局选项__（并非所有8个都出现）  

- __结局选项列表__（各结局条件详见《玩家变量与结局条件表》中的触发条件）：
	- 若 SYNC > 60 ∧ HUMAN > 50 → 显示“新伊甸”
	- 若 INCURSION > 60 ∨ HUMAN < 30 → 显示“遗忘之约”
	- 若 SYNC > 80 ∧ COUNCIL > 60 → 显示“神性监狱”
	- 若 ECHO > 70 ∧ HUMAN < 40 → 显示“统一寂静”
	- 若 COUNCIL > 70 → 显示“循环继续”
	- 若 PURIFY > 60 ∧ INCURSION > 50 → 显示“净空”
	- 若 SYNC > 90 ∧ INSIGHT > 80 → 显示“先驱之路”
	- 若 INSIGHT > 80 ∧ HUMAN > 70 ∧ SYNC > 60 ∧ 未站队 → 显示“真正的救赎”（隐藏，需额外确认）
- __玩家选择后__ \-> 锁定结局，触发FIN\-03。

<a id="四节点间的依赖与分支管理"></a>## 四、节点间的依赖与分支管理

- __线性节点__（链式）：玩家必须按顺序经历，无法跳过，例如 PRO\-01 → PRO\-02 → 01\-01 …
- __条件节点__（菱形）：只有满足前序选择才出现，例如 01\-03（小胖的秘密）只有在玩家夜晚选择跟踪时才解锁。
- __可选节点__（矩形带虚线）：不出现不影响主线，例如 01\-05（回声\-7初现）只在安装了加密模块后出现。
- __互斥节点__：例如 02\-06（卡伦的结局）会根据02\-05的选择而呈现不同内容，多个分支内容互斥。

<a id="五ai生成时的节点约束总结"></a>## 五、AI生成时的节点约束总结

当玩家进入一个节点时，AI生成对话前需注入：

- 该节点的__必达信息__（作为系统指令：必须提及这些点）
- NPC该时刻的__记忆摘要__（信任值、前序交互等）
- 该节点允许的__分支选项__（提供给用户选择）

AI可以自由组织语言、添加表情描述、依角色性格调整措辞，但不能遗漏必达信息、不能创造新的可选分支（除非是临时NPC）。
